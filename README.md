## Executando o projeto localmente
1. Vá ao diretório birds-manager

    ```
    cd birds-manager
    ```
 
2. Inicialize o json-server
    ```
    npm run json:server
    ```

3. Inicialize o projeto
    (Na mesma pasta, porém em outra aba do terminal)
    ```
    npm start
    ```
